# Surviving VSCode

## Edit
| Shortcut        | Result        |
| --------------- |:--------------|
|<kbd>CTRL</kbd>+<kbd>TAB</kbd> | show history of opened files |

## View
| Shortcut        | Result        |
| --------------- |:--------------|
|<kbd>CMD</kbd>+<kbd>B</kbd> | hide/show the Explorer |

## Markdown Edit
| Shortcut        | Result        |
| --------------- |:--------------|
|<kbd>CMD</kbd>+<kbd>K</kbd> <kbd>V</kbd> | Preview markdown |
